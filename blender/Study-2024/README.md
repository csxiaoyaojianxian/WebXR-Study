## 1. 基本操作

`shift` + `c`  游标回到世界中心

`ctrl` + `鼠标移动`  吸附网格移动

## 2. 建模操作

选中相连元素：选中点线面 + `L` 

循环选择：`ctrl` + 双击

## 3. 十大建模操作

### 3.1 四大：挤出/向内挤出/倒角/循环切割



### 3.2 六小：合并/断开/填充/切刀/桥接/分离



## 4. 建模马拉松

蘑菇：简易形变(蘑菇杆) / 吸附(表面颗粒)

树桩：衰减(苔藓) / 分离(苔藓) / 拉平(底部，sz0) / LoopTools圆环(分叉)

花朵：旋绕工具

树干：缩裹修改器

龟背竹：拓扑(E) / 合并(M) / 断开(V) / 切刀(K)



## 5. 晶格形变

用于控制区域形变，如制作松叶

`shift` + `A` 创建晶格，包住目标对象，`cmd` + `P` 绑定晶格形变

设置晶格细分，进入编辑模式后可以调节晶格形变，影响目标对象形变
